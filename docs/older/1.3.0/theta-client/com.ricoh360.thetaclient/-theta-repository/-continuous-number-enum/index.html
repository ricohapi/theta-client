ContinuousNumberEnum

enum ContinuousNumberEnum : Enum<ThetaRepository.ContinuousNumberEnum>

Number of shots for continuous shooting. It can be acquired by camera.getOptions.

For RICOH THETA X

  • 11k image: Maximum value 8

  • 5.5k image: Maximum value 20

Depending on available storage capacity, the value may be less than maximum.
